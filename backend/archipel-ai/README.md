# Archipel AI

## Installation

1. install python and [poetry](https://python-poetry.org/docs/)
2. move your terminal here

```bash
$ cd backend/archipel-ai
```

3. install dependencies

```bash
$ poetry install
```

4. run serve

```bash
$ poetry run serve
```

### Interpreter + venv

- if no python interpreter is set automatically
- set interpreter to .venv folder in /backend/archipel-ai/.venv created by poetr
